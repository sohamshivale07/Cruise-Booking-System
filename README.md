# Cruise-Booking-System
A Cruise Booking System developed using Python and MySQL.

## Overview
The Cruise Booking System is a Python-based application that allows users to book cruise seats, manage reservations, and store booking information in a MySQL database.

## Features
- User Registration
- Cruise Booking
- Seat Reservation
- Booking Management
- MySQL Database Integration

## Technologies Used
- Python
- MySQL
- SQL

## Database
Import the `seats.sql` file into MySQL before running the project.

## How to Run
1. Clone this repository.
2. git clone [https://github.com/sohamshivale07/Cruise-Booking-System.git](https://github.com/sohamshivale07/Cruise-Booking-System.git)
cd Cruise-Booking-System
3. Import `seats.sql` into MySQL.
4. Configure your database credentials in the Python file.
5. Run the project:
   ```bash
   python main.py
